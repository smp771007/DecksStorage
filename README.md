# 使用方式

  - 複製牌組代碼後，於牌組儲藏室按下**新增牌組**，便會**自動解析**並導入相關資訊，按下**新增**就完成囉。
  - **一鍵複製**牌組，快速導入遊戲內。
  - 清晰簡明的方式**快速搜尋**、**自定義分類**，再也不用找半天。
  - **default_decks**資料夾內有預設牌組，按下**操作>匯入**即可使用，感謝巴友**偶然經過的旅人**的提供
